# Arquivos

## Código Python
- `paa_backtracking.py`: Implementação do algoritmo para resolver o Problema do Caixeiro Viajante com Backtracking.


## Como Executar o Código

Para executar o código Python, utilize o seguinte comando:

```bash
python3 paa_backtracking.py
```

O programa irá:
1. Pedir para digitar o número de lojas (n) a serem visitadas
2. Irá gerar aleatoriamente as localidades das lojas
3. Gerar melhor rota paras as lojas
4. Imprimir os resultados e gera um gráfico da rota

## Requisitos

- Python 3.x
- Matplotlib

## Instalação das Dependências

```bash
pip install matplotlib
```
